## Electron js Boilerplate v1 🤖

### Getting started

``` bash
git clone https://github.com/melihs/electron-boilerplate
cd electron-boilerplate/
```

## Features
- dark mode
- show/hide sidebar
- atomic design
- quick commands

### Dev 🚀

```bash
yarn 
yarn start
```

### Make 🛠️

```bash
yarn make
```

### Optional Commands 🎁
Create components based on the selected folder.
```bash
yarn plop
```
